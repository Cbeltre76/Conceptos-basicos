1. ¿Que es un dato?

Un dato es una representación simbólica (numérica, alfabética, algorítmica, etc.) de un atributo o variable. Es la mínima unidad semántica, y por sí solo puede no tener significado relevante.

2. ¿Qué es información?

La información es el resultado del procesamiento, manipulación y organización de datos de manera que adquieren significado y utilidad para quien la consume. Es decir, los datos transformados en contexto y estructurados para ser útiles.

3. ¿Diferencia entre dato e información?

Los datos son la materia prima sin procesar, mientras que la información es el resultado del procesamiento de esos datos, dándoles significado y contexto.

4. ¿Tipos de datos?

Los datos pueden ser de diferentes tipos, incluyendo datos numéricos, datos de texto, datos booleanos, datos de fecha y hora, entre otros.

5. ¿Qué es un modelado de Datos?

El modelado de datos es el proceso de crear un modelo estructurado que representa aspectos relevantes de la realidad para una aplicación de base de datos.

6. Tipos de bases de Datos, según:

Variabilidad: Bases de datos relacionales, bases de datos NoSQL (como documentos, grafos, clave-valor, etc.). Contenido: Bases de datos de texto, bases de datos multimedia, bases de datos espaciales, etc. Modelo de base de datos: Modelo relacional, modelo de red, modelo jerárquico, etc.

7. ¿Qué es el modelo relacional?

El modelo relacional es un modelo de datos para la gestión de bases de datos en el que los datos se representan en tablas (relaciones) interconectadas.

8. ¿Describa los elementos del Diagrama entidad Relacional?

Los elementos básicos son: entidades (tablas), atributos (columnas), relaciones (conexiones entre tablas) y cardinalidad (la naturaleza y cantidad de conexiones entre entidades).

9. ¿Qué es una entidad?

Una entidad es un objeto o concepto del mundo real que se puede distinguir claramente y del cual se puede recopilar información. En un contexto de base de datos, una entidad se representa generalmente como una tabla.

10. ¿Qué es un atributo y sus tipos?

Un atributo es una propiedad o característica que describe una entidad. Los tipos de atributos pueden ser numéricos, de texto, booleanos, de fecha y hora, entre otros.

11. ¿Qué es una relación?

Una relación es una asociación entre entidades. En el modelo relacional, una relación se representa mediante una conexión entre las tablas a través de claves primarias y foráneas.

12. ¿Qué son las llaves primarias, foránea y sus características?

Llave primaria: Es un atributo o conjunto de atributos cuyos valores identifican de manera única cada fila de una tabla. Llave foránea: Es un atributo (o conjunto de atributos) en una tabla que establece una relación con la llave primaria de otra tabla. Esta relación es utilizada para conectar datos entre tablas.

13. ¿Qué son las reglas del Negocio?

Son restricciones o directrices que definen cómo una empresa o sistema debe operar. En el contexto de la base de datos, las reglas del negocio pueden incluir restricciones de integridad, validaciones de datos, entre otros.

14. ¿Qué es la cardinalidad o mapeo?

La cardinalidad describe la naturaleza y la cantidad de la relación entre dos entidades en un modelo de datos. Puede ser uno a uno, uno a muchos o muchos a muchos, entre otros. El mapeo se refiere a cómo se establecen estas relaciones en el modelo de base de datos.
